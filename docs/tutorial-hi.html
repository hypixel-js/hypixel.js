Tutorial: Getting a player's hypixel data.

Getting a player's hypixel data.

Read Home first

To fetch a player's data, we can do the following which fetches a user's hypixel data based on their uuid.

client.players.get('16751f79-c0b1-4e53-a0b5-90d31fc1d80d').then((player) => {
  console.log(player)
})

The player class has a lot of elements.
player.achievements
player.channel
player.displayName
player.firstLogin
player.friendRequests
player.game
player.karma
player.knownAliases
player.language
player.lastLogin
player.lastLogout
player.oneTimeAchievements
player.pets
player.quests
player.rank
player.rawStats
player.uuid
player.version
Be sure to check out the player documentation before using one of these.
